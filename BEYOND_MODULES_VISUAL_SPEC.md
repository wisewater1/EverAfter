### Starfield Composition

```javascript
// Star generation
Stars: 50 total
Positions: Math.random() × 100% (x, y)
Size: 1px × 1px
Color: white (#ffffff)
Opacity range: 0.2 → 1 → 0.2
Animation: 2-5s (randomized per star)
Delay: 0-3s (randomized per star)
Effect: scale(1) → scale(1.5) → scale(1)
```

### Glow Effect Composition

```css
/* Pillar glow */
.glow-effect {
  position: absolute;
  inset: 0;

  opacity: 0.6;
```
